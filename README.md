# AI Agents - Homework 5

This repository contains the implementation and analysis for MIT's "Modeling: Multimodal AI - Homework 5: AI Agents in the Wild."

## Overview

This project involves designing and implementing an AI agent that operates in an environment, takes actions over multiple steps, and attempts to accomplish user-defined goals. The focus is on building a technically grounded agentic system and rigorously analyzing its behavior.

## Requirements

- Python 3.x
- Jupyter Notebook
- Dependencies as specified in the notebook

## Setup

1. Clone the repository
2. Create a Python virtual environment (optional but recommended)
3. Install required packages
4. Copy your environment variables to `.env`
5. Open the notebook in Jupyter

## Usage

Run the notebook cells sequentially to execute the agent implementation and analysis.

## Notes

- Environment variables are stored in `.env` and are not committed to the repository
- All image assets are organized in the `image/` folder for better organization
